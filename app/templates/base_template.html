Upcoming Updates to the GEOGLOWS ECMWF Streamflow Service (GEOGLOWS 2.0)

TDXHydro Streams
  • New streams that align with TanDEM-X.
  • 5 to 6 million stream reaches.
  • Divides world into ~125 regions (previously 13).
  • Better estimates for routing parameters and travel times.
Longer Retrospective Simulation
  • 1940 start (previously 1979).
  • Default span Jan 1st, 1990 to present.
  • Computed with less than a month lag.
Modeling Workflow Improvements
  • Faster simulations.
  • Finer temporal and spatial resoulutions.
  • Changes to file formats used for improved retrieval speed (NetCDF to Zarr).
REST API Service Improvements
  • REST API versioning.
  • OGC API Standard integration.
  • Updates to read result's new Zarr format.
  • New REST API endpoints to facilitate data retrieval.

These updates are expected to be available in April, 2024. For more information join our Google Group and Mailing List Here.
